> The Latix Project
  Layered Translation Intermediate XML

	- Change name TiX to LaTiX
	- Create new POS related projects' layout
	  (Rearrange BODZ'XML structures)
	  http://www.bodz.net/usnap/<project>/<name>

	  standard names:

		text		docuemnts
		mat		materials
		src		source code
		inc		include files
		bin		binary, release executables
		sub		sub-projects
		mod		sub-modules
		lab		test, debug, etc. laboratory utilities.
		admin		管理工具
		btw		衍生物、副产品
		etc		其它、参考链接、未归类
		.vol		发布包、历史、过期事物等等
		.current	进行中的工作

	其中 .vol 专门管理、备份（目前可先放在crit-vol的专门目录中），
	而不进入vcs。
	.current 只供临时用，也不进入vcs。


> Latix 若干缩略语

	LaTi		LaTi标准中间模型
	LaTiX		LaTi XML 文档，.ltx
	LaTiS		LaTi 的一种标准英语脚本，.lts
	LaTiB		LaTi 的标准二进制格式，运行于LTP-VM之上
	含语		LaTi 的一种标准中文脚本，.han (.含)
	LaTi Platform	LaTi 的支持平台，简称LTP
	  LTP-C		LaTi C 映射的支持库
	  LTP-C++, LTP-Pascal, LTP-VB, LTP-Java 等等
	  LTP-VM	LaTi 独立 LaTiB 虚拟机
